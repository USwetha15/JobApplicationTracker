# Job Application Tracker
A responsive web application to track and manage your job search pipeline, built with HTML, CSS and JavaScript.

## Tech Stack
* Frontend: HTML5, CSS3, JavaScript (Vanilla)
* Storage: localStorage (Browser)
* Server: Apache Tomcat v11.0
* IDE: Eclipse

## Features
* Add, Edit and Delete job applications
* Filter by status — Applied, Screening, Interview, Offer, Rejected
* Search by company, role or location
* Stats Dashboard — Total Applications, Response Rate, Offers
* Grid and List view toggle
* Priority levels — High, Medium, Low
* Data saved automatically in localStorage
* Responsive dark themed UI
